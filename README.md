#  User Authentication System using PHP & MySQL

A simple and secure **User Authentication System** built with **PHP** and **MySQL**.  
This project demonstrates basic authentication features such as **Register, Login, Dashboard, and Logout** using PHP sessions and password hashing.

---

## Features

- User Registration with form validation
- Secure User Login system
- Password encryption using `password_hash()`
- Session-based authentication
- Protected Dashboard (only accessible after login)
- Secure Logout functionality

---

## Tech Stack

- **Frontend:** HTML, CSS  
- **Backend:** PHP  
- **Database:** MySQL  
- **Server:** XAMPP / WAMP
